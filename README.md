## Library SIM800l Module for Arduino UNO
The Purpose of This library is to use the module in the easy way and transparent.

## connections & pinouts
Arduino Uno  |   Sim800l   |    Notes  
-------------|-------------|------------
+5v| (3.8v)~(4.4v)!| Power supply input
10 RX_PIN | TX |  
11 TX_PIN | RX |
2  RESET_PIN | RST| Reset Pin
GND | GND |
